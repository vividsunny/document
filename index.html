Plugin Installation

  1. All Files and documentation
  2. Installable Wordpress Plugin File

If you select to download all files and documentation ( the zip file name will be xxx-xxxxx.zip ), you have to extract it out ( to get xxx-xxxxx.zip file ) before installing it to wordpress system.

Install Plugin via Wordpress Dashboard
  1. Go to "Plugins section"
  2. Click "Add New" and select the "Upload" option
  3. Upload the "xxx-xxxxx.zip" file
  4. All Done :)
Install theme via FTP
  1. Access to the file on your server using hosting control panel or ftp editor application
  2. Go to "wp-content/plugins" folder on your wordpress installation
  3. Extract the zip file and put the "xxx-xxxxx" folder there
  4. Go to "Plugins section" to activate the theme
  5. All Done :)

Default Plugin Table

After you finished installing plugin, go to "phpmyadmin / "{DATABASE}" section, you'll see an "Default plugin table" in database.
